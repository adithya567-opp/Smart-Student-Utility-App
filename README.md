Project Title

Smart Student Utility Web App

Project Description

The Smart Student Utility Web App is a simple and interactive web application designed to help students perform common academic and logical tasks in one place. It provides multiple tools that assist students in checking grades, analyzing numbers, validating login credentials, and testing words for palindrome properties. The application focuses on usability, simplicity, and responsiveness so that students can easily access these utilities from any device.

The web application is built using HTML, CSS, and JavaScript. HTML is used to structure the content and create different sections for each tool, CSS is used to design a clean and responsive interface, and JavaScript is used to implement the logic behind each feature and make the page interactive.

The system integrates several useful student tools into a single interface, reducing the need for multiple applications. It also includes a dark mode toggle, responsive layout, and modern UI components such as cards, navigation menus, and icons to enhance the user experience.

Key Features
1. Student Grade Calculator

This feature allows users to enter a student's name and marks for subjects such as Mathematics, Physics, Chemistry, Computer Science, and English. The system calculates the total marks, average score, and assigns a grade based on the average.

2. Palindrome Checker

The palindrome checker allows users to enter a word and determine whether it reads the same forward and backward. It also displays the character count of the entered word.

3. Number Analyzer

This tool analyzes a number entered by the user and determines:

Whether the number is even or odd

Whether it is positive, negative, or zero

A series of numbers based on the input value

4. Login Validator

The login validator checks whether the entered username and password match predefined credentials and displays whether the login attempt is successful or invalid.

5. Dark Mode Toggle

Users can switch between light mode and dark mode to improve readability and user comfort.

Technologies Used

HTML – Structure of the web application

CSS – Styling, layout, responsiveness, and dark mode

JavaScript – Application logic and interactivity

Conclusion

The Smart Student Utility Web App demonstrates how multiple useful student tools can be combined into a single web platform. It highlights the use of front-end technologies to build an interactive and user-friendly interface. The project is suitable for learning fundamental web development concepts such as form handling, DOM manipulation, conditional logic, and responsive design.
